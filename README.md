# Spotify Clone
https://rishh4real.github.io/spotify-clone/

A Spotify-inspired music player homepage built with HTML, CSS, and JavaScript.

## Features

- Sidebar navigation and library section
- Featured hero banner with category switching
- Search and filter tracks by mood or type
- Playlist table with playable rows
- Queue preview for upcoming songs
- Sticky music player bar with next, previous, shuffle, and volume controls
- Browser-generated demo audio previews using the Web Audio API
- Dark and light mode toggle with saved preference
- Responsive layout for smaller screens

## Project Structure

- `index.html` contains the page layout
- `style.css` contains the complete styling
- `script.js` handles filtering, playback, and interactivity
- `assetss/` stores icons and cover images

## Run Locally

Open `index.html` in a browser.
