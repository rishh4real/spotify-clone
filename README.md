# Spotify Clone

A simple Spotify-inspired music player homepage built with HTML, CSS, and JavaScript.

## Features

- Sidebar navigation and library section
- Featured hero banner
- Quick pick playlist cards
- Trending music grid with hover play buttons
- Sticky music player bar
- Dark and light mode toggle
- Responsive layout for smaller screens

## Project Structure

- `index.html` contains the page layout
- `style.css` contains the complete styling
- `script.js` handles simple interactivity
- `assetss/` stores icons and cover images

## Run Locally

Open `index.html` in a browser.
